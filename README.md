# Airline Analysis Dashboard

## Project Overview
Developed an interactive Power BI dashboard to analyze airline operational performance and passenger trends. The dashboard provides insights into flight delays, cancellations, passenger traffic, and overall airline performance.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key Features
- Flight Performance Analysis
- Delay & Cancellation Analysis
- Passenger Trend Analysis
- Interactive Dashboard with Slicers
- KPI Cards
- Dynamic Visualizations

## Key Insights
- Identified airlines with the highest delays.
- Analyzed monthly passenger trends.
- Compared airline operational performance.
- Created interactive reports for business decision-making.
